Quick Start

Типы без хлопот

Мощный, безопасный вывод типов означает, что вам редко придется указывать типы, но все автоматически будет проверено для вас.

Лёгкое взаимодействие с JavaScript

Используйте пакеты из NPM/Yarn с минимальными хлопотами или даже просто вставляйте JavaScript-фрагменты для изучения!

Гибкий и веселый

Делайте веб-сайты, анимации, игры, серверные приложения, консольные инструменты и многое другое! Посмотрите на эти примеры, чтобы получить вдохновение.

Краткое руководство

npm install -g bs-platform
bsb -init my-first-app -theme basic-reason
cd my-first-app
npm run start

Это работает в режиме наблюдения, поэтому любые изменения в файлах будут подхвачены и скомпилированы. Читайте больше здесь!
