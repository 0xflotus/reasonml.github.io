Quick Start

Типи без клопотів

Потужне, статично безпечне наслідування типів означає, що вам рідко доведеться писати анотації для типів, так як все перевіряється автоматично для вас.

Проста взаємодія з JavaScript

Ви можете без проблем використовувати NPM/Yarn пакети, або, навіть, просто вставити справжній JavaScript код під час навчання!

Гнучке і прикольне

Створюйте сайти, анімації, ігри, сервери, консольні програми і багато іншого! Щоб надихнуться, подивіться на ці приклади.
