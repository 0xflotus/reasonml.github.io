編輯器外掛

確認你已經從上一個章節安裝 reason-cli。

Reason 的天性給編輯器有極大的支援。大多數編輯器外掛，至少提供了：

  • 型別顯示。
  • 透過 refmt 自動編排程式碼。
  • 錯誤 & 警告顯示。
  • 語法高亮顯示。
  • 型別驅動自動補完。
  • 跳至定義。

以及其他更多功能。請見，例如，VSCode 外掛的功能章節！

官方支援編輯器

  • VSCode：推薦。
  • Atom
    • 請安裝相關套件 apm install language-reason linter linter-refmt reason-refmt。
    • 或者，如果使用 Nuclide，預設已經支援。
    • 真正 JavaScript 精神，另一個 Atom Reason 外掛！ 事實上，這可能是未來的 Atom Reason 外掛。 如果感到不安，請用它吧！
  • Vim
  • Emacs
  • Sublime Text
  • IDEA

GitHub reasonml-editor 社群存放大多數的外掛。 如果你想要加入你最愛的編輯器外掛，請發起一個 PR！
