字串 & 字元

字串

Reason 字串使用雙引號分隔（單引號保留供字元型別使用）。

let greeting = "Hello world!";
let multilineGreeting = "Hello
 world!";

字串中特殊字元需要脫逸（escaped）：

let oneSlash = "\\";

串接字串請用 ++：

let greetings = "Hello " ++ "world!";

帶引號的字串

字串有個特殊句法允許

  • 如同往常的多行字串
  • 不須額外脫逸字元
  • 特殊前處理（pre-processors）的鉤點（hooks）
let greetingAndOneSlash = {|Hello
World
\
Hehe...
|};

類比來說，就像是 JavaScript 的反引號（backtick）字串插值（interpolation），只是不需要額外的脫逸特殊字元，以及沒有內建變數插值。 儘管如此，你可以恢復後者的功能，如同 BuckleScript 所作：

let world = {js|世界|js}; /* 支援 Unicode 字元 */
let helloWorld = {j|你好，$world|j}; /* 支援 Unicode 以及變數插值 */

BuckleScript 的特殊前處理將找尋字串周圍的 js 及 j 標記，並轉換成其他東西。

用法

標準函式庫中可以找到更多有關字串操作。 若是編譯成 JS，請見類似的 Js.String API，位於 BuckleScript API 文件。 由於 Reason 字串對應到 JavaScript 字串，你可以混合 & 相配兩者標準函式庫的字串操作。

提示 & 技巧

https://twitter.com/jusrin00/status/875238742621028355

現在，你擁有了敘述能力的型別系統，在非型別語言中， 經常透過下列方式來覆載字串的意義：

  • 獨特 ID：var BLUE_COLOR = "blue"
  • 資料結構的識別子：var BLUE = "blue"; var RED = "red"; var colors = [BLUE, RED]
  • 物件欄位名稱：person["age"] = 24
  • 列舉：if (audio.canPlayType() === 'probably') {...} (ಠ_ಠ)
  • 其他你會感到可怕的瘋狂樣式，當你習慣 Reason 的替代方案後。

越多覆載糟糕的字串型別，越少型別系統（或隊友）能幫你！ Reason 提供簡潔、快速且可維護型別 & 資料結構，可取代上述使用情境（例如變異，稍後章節會說明）。

底層原生編譯中，Reason 字串編譯成簡單表現法，其效能十分容易分析，某些擴張情境下需要手動調校效能。 舉例來說，天真地串接字串像是 "hi " ++ "how " ++ "are " ++ "you?" 並不需要分配中介字串 "are you?" 及 "how are you?"（儘管這個簡單案例來看可以直接最佳化成一個字串）。 在此例中，偏好使用 String.concat。 某種程度上，學校學到的傳統執行期分析終於能夠再度派上用場。

Under JavaScript compilation, a Reason string maps to a JavaScript string and vice-versa, so no such above concern or analysis opportunities apply.

設計決策

帶引號字串不需要脫逸字元的功能開啟優美 DSLs 像是 正規表達式：

let r = Str.regexp({|hello \([A-Za-z]+\)|});

而不是

let r = Str.regexp("hello \\([A-Za-z]+\\)");

儘管在 JS 編譯中，你需要改成使用 [%bs.re] 及 Js.Re ，不可使用 Str。

這裡可以看見 Reason/OCaml 強調簡單過於聰明，原生字串實作十分了當。 過於複雜的字串實做有時容易玩火自焚。

字元

Reason 對單一字母有另外型別：

let firstLetterOfAlphabet = 'a';

注意: 字元不支援 Unicode 或 UTF-8。

提示 & 技巧

為了速度考量，字元編譯成整數範圍從 0 到 255，你可以樣式匹配（稍後章節會說明）：

let isVowel = (theChar) =>
  switch (theChar) {
  | 'a'| 'e'| 'i'| 'o'| 'u'| 'y' => true
  | _ => false
  };

若要將字串轉換字元，使用 "a".[0]。轉換字元成字串，使用 String.make(1, 'a')。
