快速上手

npm install -g bs-platform  # 提供 BuckleScript 編譯器
bsb -init my-first-app -theme basic-reason  # 新增專案 my-first-app/
cd my-first-app
npm run build  # 使用  BuckleScript 編譯器編譯成 src/Demo.bs.js
node src/Demo.bs.js  #  使用 NodeJS 執行程式碼

你會看見 "hello world" 輸出：

> node src/Demo.bs.js
Hello, BuckleScript and Reason!

我們使用 BuckleScript 編譯器將 Reason 編譯成 JavaScript，接著使用 NodeJS 執行 JavaScript。 執行編譯器之後，接下來就是熟悉的 JavaScript 流程（例如執行 node 或者瀏覽器）。

開發過程中，並不是每次都執行 npm run build ，而是執行 npm run start 啟動監看，一旦檔案更動便重新編譯。
