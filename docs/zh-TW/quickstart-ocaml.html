快速上手

注意：原生工作流程目前正在緩慢開發中。

# 在 OSX，透過 Homebrew 安裝 opam：
brew update
brew install opam
# 在 Linux，請參考這裡（需要 opam >= 1.2.2）：http://opam.ocaml.org/doc/Install.html
opam init
# **注意**：加入下列幾行到 ~/.bashrc 或 ~/.zshrc 檔；每次 shell 啟動時都必須使用
eval $(opam config env)
opam update
opam switch 4.02.3

接著複製範例 ReasonNativeProject 儲存庫，接下來就可以開始了！
