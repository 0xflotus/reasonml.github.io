빠른 시작

중요: 네이티브 워크플로우는 현재 개발중에 있습니다.

# macOS 에서는, Homebrew 를 이용하여 opam 을 설치하세요.
brew update
brew install opam
# Linux 에서는, 이 사이트를 참고해 주세요. (opam 버전 >= 1.2.2) - http://opam.ocaml.org/doc/Install.html
opam init
# **중요**: ~/.bashrc 이나 ~/.zshrc 에 공백을 추가해야합니다. 매 shell 시작때마다 필요합니다.
eval $(opam config env)
opam update
opam switch 4.02.3

그리고 저희들의 ReasonNativeProject 레포지토리를 클론하면, 이제 마음대로 하셔도 됩니다!
