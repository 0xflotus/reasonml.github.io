Editor Plugins

请确认您已经在上一节安装过了 reason-cli。

Reason 天然地对编辑器有很好的支持。我们大多数编辑器插件至少提供了：

  • 展示的种类
  • 通过 refmt 进行代码格式化.
  • 错误 & 警告显示。
  • 语法高亮。
  • 类型驱动自动补全。
  • 跳转到定义处。

以及其它功能。例如，VSCode 插件的 功能章节 ！

官方支持的编辑器

  • VSCode：推荐。
  • Atom
    • 请使用命令apm install language-reason linter linter-refmt reason-refmt安装相关的包。
    • 或者，如果您使用Nuclide，Reason 已经默认支持。
    • 真正的 JavaScript 精神，这里是另外一个Atom Reason 插件！ 事实上，这可能是未来事实的 Atom Reason 插件。 如果您感觉有些冒险，也请使用它。
  • Vim
  • Emacs
  • Sublime Text
  • IDEA

GitHub reasonml-editor 社区主持着很多这种插件。 如果你想要把你喜欢的编辑器插件加到这里，请给我们发一个 pull request！
