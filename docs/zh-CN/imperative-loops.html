命令式循环

For 循环

For循环是从起始值到结束值(包括结束值) 的循环。

for (myBinding in startValue to endValue) {
  /* use myBinding here */
};

如果不需要，则可以省略startValue和endValue周围的括号。

let xStart = 1;
let xEnd = 3;

/* prints: 1 2 3 */
for (x in xStart to xEnd) {
  print_int(x);
  print_string(" ")
};

使用downto，则可以使 for 循环计数递减。

for (myBinding in startValue downto endValue) {
  statements
};

let xStart = 3;
let xEnd = 1;

/* prints: 3 2 1 */
for (x in xStart downto xEnd) {
  print_int(x);
  print_string(" ")
};

While 循环

While循环在某些条件为真时执行代码块。while循环的形式包含一个表达式，即要测试的条件。

while (testCondition) {
  statements
};

提示& 技巧

在Reason中没有循环跳出break关键字(函数也没有return)。 但是，我们可以通过使用 可变绑定 来轻松地中断 while 循环。

Random.self_init();

let break = ref(false);

while (! break^) {
  if (Random.int(10) === 3) {
    break := true
  } else {
    print_endline("hello")
  }
};

我们对于是否使用map/filter/reduce循环没有的喜好。 因为不同的使用场合，有不同的需求，不能统一而定。
