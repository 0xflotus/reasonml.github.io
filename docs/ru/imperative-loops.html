Императивные циклы

Циклы for

Циклы for позволяют итерироваться от начального до конечного (включительно) значений.

for (myBinding in startValue to endValue) {
  /* используйте myBinding тут */
};

Скобки вокруг startValue и endValue могут быть пропущены, если необязательны.

let xStart = 1;
let xEnd = 3;

/* prints: 1 2 3 */
for (x in xStart to xEnd) {
  print_int(x);
  print_string(" ")
};

Вы можете итерировать for в обратную сторону, используя downto.

for (myBinding in startValue downto endValue) {
  statements
};

let xStart = 3;
let xEnd = 1;

/* prints: 3 2 1 */
for (x in xStart downto xEnd) {
  print_int(x);
  print_string(" ")
};

Циклы while

While loops execute a code block while some condition is true. The form of a while loop includes a single expression, the condition to test.

while (testCondition) {
  statements
};

Советы и трюки

В Reason нет способа остановить цикл в середине (break в других языках) и нет способа сделать возврат из середины функции (return). However, we can break out of a while loop easily through using a mutable binding.

Random.self_init();

let break = ref(false);

while (! break^) {
  if (Random.int(10) === 3) {
    break := true
  } else {
    print_endline("hello")
  }
};

We don't have a preference on whether you should use a loop over map/filter/reduce. Some algorithms are better expressed with the former, and some with the latter.
