Краткое руководство

npm install -g bs-platform # Устанавливает BuckleScript компилятор
bsb -init my-first-app -theme basic-reason # Создает проект my-first-app/ 
cd my-first-app 
npm run build # Компилирует в src/Demo.bs.js при помощи BuckleScript
node src/Demo.bs.js # Запускает Демо используя NodeJS

Мы должны увидеть приветствие:

> node src/Demo.bs.js
Hello, BuckleScript and Reason!

Мы используем BuckleScript для компиляции Reason в JavaScript, а потом используем NodeJS для запуска JavaScript. После запуска компилятора, все, что осталось, это знакомый JavaScript процесс (например запуск в node или браузере).

При разработке, вместо запуска npm run build каждый раз, запустите npm run start, чтобы запустить процесс, который будет перекомпилировать при изменении файлов.
