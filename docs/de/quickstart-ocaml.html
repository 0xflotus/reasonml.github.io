Schnellstart

**Hinweis: der native Workflow befindet sich unter starker Bearbeitung **.

# In OSX, installiere opam via Homebrew:
brew update
brew install opam
# In Linux, siehe hier (du brauchst opam >= 1.2.2): http://opam.ocaml.org/doc/Install.html
opam init
# **Hinweis**: trage die Zeile unten in deine ~/.bashrc oder ~/.zshrc ein; sie wird beim Start einer neuen Shell gebraucht
eval $(opam config env)
opam update
opam switch 4.02.3

Klone dann unser beispielhaftes ReasonNativeProject Repository und es kann losgehen!
