Integer & Float

Integers

32-bits, tronqués lorsque cela est nécessaire. Raison fournit les opérations habituelles sur eux : +, -, *, /, etc.

Utilisation

Voir le module Int32 de la librairie standard. Pour la compilation de JS, voir Js.Int.

Conseils & astuces

Attention lorsque vous faîtes des binds avec des nombres JavaScript ! Les plus longs peuvent être tronqués. Bindez le number JavaScript à un float à la place.

Floats

Le float exige d'autres opérateurs : +., -., *., /., etc. Exemple : 0,5 +. 0,6.

Utilisation

Il n'y a pas de module Float dans la librairie standard actuelle. Pour la compilation JavaScript, voir Js.Float.

Décisions de Conception

"Why the heck can't I just use an overloaded + for both int and float?"

There's a lot of history to this, but one day we'll be able to. Let's hang in there =)

En outre, les floats sont plutôt spéciaux en Reason/OCaml natif. Vérifiez ici si avez envie d'apprendre des optimisations plutôt intéressantes !
