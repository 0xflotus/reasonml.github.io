Librairies

La source principale de librairies est l’Index de Paquets Reason (Redex), qui conserve une liste des paquets disponibles, avec possibilité de la parcourir ou d’y faire des recherches. Il contient aussi une section pour les paquets non publiés, ce qui signifie généralement qu’ils ne sont pas terminés ou expérimentaux mais peuvent toutefois répondre à votre besoin.

Les paquets qui ne sont pas encore dans le redex peuvent être trouvés dans les paquets NPM ou sur GitHub. Les librairies pour lesquelles un wrapper à déjà été conçu avec BuckleScript sont habituellement préfixées par bs-, par exemple bs-director. Voici une recherche pour toutes les bibliothèques relatives à BS sur NPM.

Si vous ne trouvez pas ce que vous cherchez, sachez que vous n’avez pas systématiquement besoin d'un wrapper pour utiliser une librairie JavaScript. La plupart des types de données JavaScript, comme les tableaux ou les objets, s’utilisent très bien dans Reason, et vice-versa. Vous avez aussi accès à la familière API JavaScript. Pensez aussi à jeter un œil aux sections « Interopérabilité » et « Conversion depuis JavaScript » ici !
