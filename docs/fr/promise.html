Promise

Reason a un supporte par défaut les promises JavaScript via BuckleScript. Les 3 fonctions dont vous avez généralement besoin sont :

  • Js.Promise.resolve: 'a => Js.Promise.t('a)
  • Js.Promise.then_: ('a => Js.Promise.t('b), Js.Promise.t('a)) => Js.Promise.t('b)
  • Js.Promise.catch: (Js.Promise.error => Js.Promise.t('a), Js.Promise.t('a)) => Js.Promise.t('a)

En outre, voici comment créer une promesse JavaScript en Raison:

  • Js.Promise.make: ( ( ~resolve: (. 'a) => unit, ~reject: (. exn) => unit ) => unit ) => Js.Promise.t('a)

Cette signature de type signifie que make prend un callback qui reçoit 2 arguments nommés, resolve et reject. Les deux arguments sont eux-mêmes des callbacks non-curryfiés (ils ont un point). make retourne la promesse crée.

Utilisation

Via l'opérateur pipe (voir "Opérateurs de composition" dans la section modules pervasifs) :

let myPromise = Js.Promise.make((~resolve, ~reject) => resolve(. 2));

myPromise
|> Js.Promise.then_(value => {
     Js.log(value);
     Js.Promise.resolve(value + 2);
   })
|> Js.Promise.then_(value => {
     Js.log(value);
     Js.Promise.resolve(value + 3);
   })
|> Js.Promise.catch(err => {
     Js.log2("Failure!!", err);
     Js.Promise.resolve(-2);
   });

Note : nous pourrions offrir une syntaxe dédiée pour les promises JS (async/await) dans le futur. Restez à l'écoute !
