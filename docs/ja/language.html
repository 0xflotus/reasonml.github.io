Language basics

Reason 学習の旅をここから始めましょう！ポイントをいくつか:

  • 多くのセクションで「ヒントとコツ」、「設計方針」セクションがあります。これらは補足事項なので自由に読み飛ばして構いませんが、回り道をすることで、ほんのちょっとおいしい情報が学べますよ！
  • 既存の JS ライブラリをバインドする必要がある場合は、BuckleScript の FFI があなたのお供になります。
  • つまづく場合は、私たちのコミュニティで質問してください。もしくは右側にある "Suggest an edit" ボタンををクリックして pull request を送ってください。

お楽しみください！
