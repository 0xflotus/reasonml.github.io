Lazos imperativos

Bucles For

Los bucles for se repiten empezando por un valor inicial hasta (e incluyendo) alcanzar el valor final.

for (myBinding in startValue to endValue) {
  /* usa myBinding aquí */
};

Los paréntesis alrededor de startValue y endValue pueden omitirse si no son necesarios.

let xStart = 1;
let xEnd = 3;

/* imprime: 1 2 3 */
for (x in xStart to xEnd) {
  print_int(x);
  print_string(" ")
};

Puedes hacer al bucle for contar en la dirección opuesta usando downto.

for (myBinding in startValue downto endValue) {
  /* declaraciones */
};

let xStart = 3;
let xEnd = 1;

/* imprime: 3 2 1 */
for (x in xStart downto xEnd) {
  print_int(x);
  print_string(" ")
};

Bucles While

Los bucles while ejecutan un bloque de código mientras una condición es verdadera. La forma de un bucle while incluye una única expresión, la condición a comprobar.

while (testCondition) {
  statements
};

Consejos y Trucos

En Reason no existe una palabra clave break (ni siquiera return para volver anticipadamente de funciones). Sin embargo, podemos interrumpir un bucle while fácilmente usando un binding mutable.

Random.self_init();

let break = ref(false);

while (! break^) {
  if (Random.int(10) === 3) {
    break := true
  } else {
    print_endline("hola")
  }
};

No tenemos ninguna preferencia sobre si debes utilizar un bucle en vez map/filter/reduce. Algunos algoritmos se expresan mejor con bucles y algunos de la otra manera.
