Currying

JavaScriptReason
let add = a => b => a + blet add = (a, b) => a + b

JavaScript y Reason ambos soportan currying, pero currying en Reason viene integrado y optimizado para evitar llamadas y asignaciones intermedias de funciónes, siempre que sea posible.

Bloques

En Reason, "expresiones de secuencia" se crean con {} y se evalúan a su última declaración. En JavaScript, esto puede ser simulado a través de una expresión de función invocada inmediatamente (ya que el cuerpo de las funciones tienen su propio ámbito local).

JavaScript 

let res = (function() {
  const x = 23;
  const y = 34;
  return x + y;
})();
