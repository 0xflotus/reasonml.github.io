Regalos extras

Extensiones del Navegador: Reason-tools

Reason-tools te permite intercambiar rápidamente entre la sintaxis de OCaml y Reason cuando estas hojeando tutoriales o documentación escritos en cualquiera de las dos sintaxis.

BuckleScript

Vas a ver BuckleScript extensivamente durante el resto de la documentación, dado que es el motor que mueve nuestra compilación JavaScript. Sus binarios globales se instalan mediante npm install -g bs-platform.

Otras Utilidades

La instalación global que has realizado en la sección anterior viene con algunas herramientas extra, explicadas aquí.

Refmt

refmt ("Reason format") es el binario que provee el formateo automático en nuestros editores de código. Lee tu código y lo devuelve elegantemente formateado. También sirve para convertir la sintaxis de Reason a OCaml y viceversa. La comunidad de Reason lo usa para conseguir consistencia en el estilo del código en todos los proyectos, y evitar el trabajo manual y los debates de estilo.

It responsively formats the code based on the editor width. In other words, it doesn't just naively break to the next line at a certain characters limit; it solves the layout constraints and arranges your code accordingly.

refmt puede ser usado directamente en el terminal. Por ejemplo, para formatear tu código fuera del editor, ejecuta refmt --in-place myFile.re. Usa refmt --help para conocer todas las opciones.

Merlin

Merlin es el motor subyacente que proporciona sugerencias de tipos, refactorización, errores en tiempo real, salto a la definición, etc. para nuestros editores. El binario de linea de comandos se llama ocamlmerlin, aunque no necesitas ejecutarlo manualmente (los editores lo ejecutan automáticamente).

Para configurar Merlin y que entienda tu proyecto, tienes que crear un archivo .merlin en la raíz del proyecto (aquí). Para el flujo de trabajo con JS, esta configuración es automáticamente generada para ti por bsb de BuckleScript.

REPL

Reason viene con un REPL llamado rtop que, una vez invocado, te deja evaluar tu código interactivamente. Proporciona autocompletado inteligente, basado en tipos.

Usa #quit; para cerrar tu sesión.

Ten en cuenta que rtop por ahora no se puede usar fácilmente con paquetes y externals. Recomendamos evaluar el código usando nuestra utilidad Try.

re:bench

re:bench es una utilidad de evaluación de rendimiento online. Soporta casos de pruebas escritos en Reason, OCaml y JavaScript y permite compartir resultados de rendimiento a través de URLs únicas. Está compilado a JS usando BuckleScript.

Redex

Redex: el registro de paquetes de Reason.

ocamlc, ocamlopt, ocamlrun

ocamlc y ocamlopt son los compiladores de Ocaml esenciales.
