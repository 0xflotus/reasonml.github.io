Librerías

The main source of libraries is the Reason Package Index (Redex), which keeps a curated list of available packages, and is both searchable and browsable. También tiene una sección para paquetes no publicados, que generalmente significa que no están acabados o son experimentales pero todavía podrían servirte.

Los paquetes que aún no han sido indexados por redex podrían ser encontrados en npm o GitHub. Libraries that have already been wrapped with BuckleScript are customarily prefixed with bs-, e.g. bs-director. Aquí tienes una búsqueda en npm de todas las librerías BS relacionadas.

If you can't find what you're looking for: sometimes you don't need a wrapper to use a JS library. La mayoría de tipos de datos de JS, tales como arrays y objetos, se mapean limpiamente a Reason y viceversa. También tienes acceso a la conocida API de JS. ¡Asegúrate también de consultar las secciones de "Interoperabilidad" y "Convirtiendo desde JS" aquí!
