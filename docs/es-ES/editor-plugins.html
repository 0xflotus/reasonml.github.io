Plugins para el editor

Asegúrate haber instalado reason-cli en la sección anterior.

La naturaleza de Reason confiere un gran soporte para editores. La mayoría de nuestros plugins para editores proveen como mínimo:

  • Visualización de tipos.
  • Formateo de código programático mediante refmt.
  • Visualización de errores y advertencias.
  • Resaltado de sintaxis.
  • Autocompletado basado en tipos.
  • Accesso rapido a la definición.

Y otras características. Vea, por ejemplo, de nuestro plugin para VSCode la ¡sección de características!

Editores soportados oficialmente

  • VSCode: recomendado.
  • Atom
    • Por favor instala los paquetes relacionados mediante apm install language-reason linter linter-refmt reason-refmt.
    • Alternativamente, si usas Nuclide, el soporte para Reason viene por defecto.
    • Fiel al espíritu de JavaScript, ¡aquí tienes otro plugin más para Atom! En realidad, este es el potencial plugin Reason de facto para Atom. Si te sientes aventurero, ¡por favor pruébalo!
  • Vim
  • Emacs
  • Sublime Text
  • IDEA

La comunidad reasonml-editor de GitHub tiene la mayoría de estos plugins. Si quieres añadir tu favorito plugin para editor, ¡mándanos un pull request!
