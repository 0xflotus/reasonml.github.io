Currying

JavaScriptReason
let add = a => b => a + blet add = (a, b) => a + b

Ambos JavaScript e Reason suportam currying, entretanto em Reason currying é embutido na linguagem e otimizado de maneira a evitar alocações & chamadas de funções intermediárias, quando é possível.

Blocos

Em Reason, "sequências de expressões" são criadas com {} e são avaliadas com o valor de sua última declaração. Em JavaScript, isso pode ser emulado com uma expressão de função imediatamente chamada (immediately-invoked function expression), dado que corpos de função possuem seu próprio escopo local.
