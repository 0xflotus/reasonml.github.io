Plugins de Editores

Tenha certeza que você instalou o reason-cli da seção anterior.

A natureza do Reason tende a ter um grande suporte a editores. A maioria dos nossos plugins para editores fornece no mínimo:

  • Exibição de tipos.
  • Formatação programática de código através de refmt.
  • Erros & exibição de avisos.
  • Destaque de sintaxe.
  • Autocomplete dirigido pelo tipo.
  • Ir para definição.

E outras características. Veja, por exemplo, a seção de recurso do nosso plugin para VSCode!

Editores Oficialmente Suportados

  • VSCode: recomendado.
  • Atom
    • Por favor, instale os pacotes apm install language-reason linter linter-refmt reason-refmt.
    • Alternativamente, caso você use Nuclide, o suporte ao Reason vem por padrão.
    • Fiel ao espírito de JavaScript, aqui está mais um plugin Reason para o Atom! Na verdade, esse é o unico plugin prospero do Reason para o Atom. Se você se sentir aventureiro, por favor, contribua!
  • Vim
  • Emacs
  • Sublime Text
  • IDEA

A comunidade do reasonml-editor no GitHub hospeda a maioria desses plugins. Caso você queira adicionar o seu editor favorito aqui, envie um pull request!
