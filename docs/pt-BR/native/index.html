Nativo

Atualmente usamos por padrão o workflow do OCaml para compilar para nativo, no entanto existe um trabalho para trazer o suporte nativo a compilação para o construtor do BuckleScript.

Nota: algumas partes do fluxo de trabalho nativo provavelmente não funcionam no Windows. O fluxo de trabalho nativo é atualmente um trabalho em andamento, pois atualmente estamos concentrados em polir o fluxo de trabalho JS. Contribuições são bem-vindas!

For native package management, we're testing Esy.
