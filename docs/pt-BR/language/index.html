Inicie sua jornada de estudos do Reason aqui! Alguns pontos a serem considerados:

  • Muito de nossas sessões possuem as sessões "Dicas" e "Decisões de design". Elas são de leitura opcional; sinta-se livre de pula-los, mas volte mais tarde para aprender coisas a mais!
  • Caso você precise dar bind numa biblioteca JS, BuckleScript's FFI pode te ajudar.
  • Se você sentir dificuldade, faça perguntas para a nossa comunidade e/ou envie um pull request clicando no botão "Sugerir uma edição" à direita.
