Destructuring

"Destructuring" é uma maneira visualmente concisa de extrair campos de uma estrutura de dados. Você pode usar desestruturação em qualquer lugar que você normalmente use uma variável.

Uso

O seguinte código vincula as variáveis: ten = 10, twenty = 20

let someInts = (10, 20);
let (ten, twenty) = someInts;

O seguinte código vincula as variáveis: name = "Guy", age = 30

type person = {name: string, age: int};
let somePerson = {name: "Guy", age: 30};
let {name, age} = somePerson;

Quando você extrai os campos, você pode opcionalmente renomea-los. O seguinte vincula estes em vez disso: n = "Guy", a = 30.

let {name: n, age: a} = somePerson;

Destructuring também permite tipos de anotação.

let (ten: int, twenty: int) = someInts;
let {name: (n: string), age: (a: int)} = somePerson;

Também é possível usar o "Destructing" para os argumentos rotulados de uma função.

type person = {
  name: string,
  age: int
};

let someFunction = (~person as {name}) => {
  /* you can use `name` here */
};

let otherFunction = (~person as {name} as thePerson) => {
  /* you can use both `name` and the whole record as `thePerson` here */
};

Dicas & Truques

Desestruturação ("Destructuring") pode fazer seu código ficar muito mais conciso sem exigir que você nomeie variáveis intermediárias. Use, mas não abuse disto, pois o excesso pode tornar seu código excessivamente aninhado & lacônico.
