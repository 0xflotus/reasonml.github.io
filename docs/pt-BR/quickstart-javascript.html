Introdução

npm install -g bs-platform
bsb -init my-first-app -theme basic-reason

Então, execute-o como o usual:

cd my-first-app
npm run start

Desta maneira roda em modo supervisão, em que qualquer mudança nos arquivos será detectada e compilada automaticamente. Isto é tudo!

  • Leia mais sobre como compilamos para JavaScript usando nosso projeto parceiro, BuckleScript.
  • Alternativamente, para iniciar uma aplicação ReasonReact, tente bsb -init my-react-app -theme react.
  • Vá para Setup do Editor para configurar o plugin de Reason para o editor de sua preferência!
