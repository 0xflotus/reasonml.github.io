Introdução

Observação: o fluxo de trabalho nativo é mais intenso que o trabalho em andamento.

# No OSX, instale opam via Homebrew
brew update
brew install opam
#No Linux, veja aqui (você precisará opam >= 1.2.2):
http://opam.ocaml.org/doc/Install.html
opam init
#**Nota**: adicione a linha abaixo ao seu ~/.bashrc ou ~/.zshrc também; é necessário em todos inícios de shell
eval $(opam config env)
opam update 
opam switch 4.02.3

Então clone nosso repositório exemplo ReasonNativeProject, e você estará pronto para começar!
